## Git

Local에서 만들고 Remote repo로 PUSH하기 보다는 Remote를 clone하는 게 더 효율적이다.

클론하고 커밋을 한 후 로그를 확인하면 2개가 있는 것을 확인할 수 있다. 이는 clone 하면서 commit history를 함께 가져왔기 때문에 initial commit이 생겼기 때문이다.
